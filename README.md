# Customer Support Ticket Classification (NLP)

This project classifies customer support tickets into categories using Natural Language Processing techniques.

## 🔧 Techniques Used
- TF-IDF Vectorization
- Support Vector Machine (SVM)
- Text Preprocessing

## 📊 Result
Achieved ~50% accuracy on real-world customer support dataset after proper preprocessing and model building.

## 📁 Dataset
Customer Support Tickets Dataset (Kaggle)

## 💡 Key Learning
Handled noisy real-world text data and improved model performance using proper preprocessing and pipeline techniques.
